# sistema-estagiarios-aeroportos
Sistema de Controle de Estagiários para Aeroportos de Maputo Descrição do Projeto Este repositório contém o código-fonte para um sistema web de controle de estagiários desenvolvido para os Aeroportos de Maputo. O sistema permite o cadastro de estagiários, registro de presenças, atribuição de tarefas, geração de relatórios e gerenciamento de usuários com foco em segurança e usabilidade. O desenvolvimento é baseado nos módulos do Certificado Vocacional Nível 5 (CV5) em Programação de Aplicações Web, conforme o documento "CV5-em-Programacao-de-Aplicacoes-Web.pdf" do PIREP (Programa Integrado de Reforma da Educação Profissional). Ele aplica conceitos como análise de sistemas (UC TIC025002), programação orientada a objetos (UC TIC025003), bases de dados (UC TIC025005), HTML/CSS/JavaScript (UC TIC025006-009), programação server-side com PHP (UC TIC025010-011) e segurança (UC TIC025012). Tecnologias Utilizadas:  Front-end: HTML5, CSS3, JavaScript (com bibliotecas como Bootstrap e jQuery para interfaces responsivas). Back-end: PHP (orientado a objetos, com PDO para acesso a dados). Banco de Dados: MySQL (relacional, otimizado para consultas eficientes). Ambiente: XAMPP (Apache + MySQL + PHP) para desenvolvimento local. Outros: Git para versionamento, UML para diagramas de análise.  Este projeto serve como prática para o módulo de projeto final (UC TIC025013) e experiência de trabalho (MO TIC025014), alinhado ao estágio nos aeroportos. Requisitos Funcionais  Cadastro de Estagiários: Registrar dados como nome, email, telefone, curso, departamento, datas de início/fim. Registro de Presença: Marcar presença diária com data/hora automática (presente/ausente). Gestão de Tarefas: Atribuir tarefas com descrição, prazo e status (pendente/concluída). Relatórios: Gerar relatórios de presenças mensais, desempenho e exportação (PDF/CSV). Autenticação: Login para supervisores/admins e estagiários (visão limitada). Notificações: Envio de emails para lembretes (usando PHPMailer).  Requisitos Não Funcionais  Responsivo para dispositivos móveis (usável em aeroportos). Seguro contra vulnerabilidades comuns (SQL injection, XSS). Documentação em português, com código comentado. Compatível com servidores Linux/Windows (Apache/IIS).  Instalação  Pré-requisitos: XAMPP instalado (versão 7+ com PHP 7.4+ e MySQL). Git instalado. MySQL Workbench (opcional para gerenciar BD).
